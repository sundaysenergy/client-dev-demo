client-dev-demo
===============

This is showing off a few things.

* Gulp
* Browser Sync
* Jade
* Less

## Usage

When installing gulp globally you may need to use sudo. `sudo npm i gulp -g`

    git clone git@github.com:sundaysenergy/client-dev-demo.git
    cd client-dev-demo
    npm i
    npm i gulp -g
    gulp
